PURPOSE: Estimation of the central location ('mean') of a distribution, resistant for outliers and robust for a broad range of non-Gaussian underlying populations.

INPUT: vector = distribution in vector form

OUTPUT: mean = central location of the distribution STDDEV = scale of the distribution ci_mean[0],ci_mean[1]= confidence interval for the mean (default 95%) ci_std[0],ci_std[1]= confidence interval for the mean (default 95%)

KEWORDS:

itmax=10 = maximum number of iterations used in determination of the central location; default 10 fracmn=[] = minimum fractional change in median absolute deviation (MAD), used as convergence criterion in iteration; default 0.03*sqrt(0.5/(n-1)), with n number of vector elements eps=1e-24 = small number: iteration is stopped if MAD < EPS; default 1.0e-24

ci_perc=95 = default confidence interval (95), 1sigma=68.4 ci_mean=True -> if True (default) the code output the confidence interval for the mean as output[2] and output[3], else ouput[2]=output[3]=0 ci_std=True -> if True (default) the code output the confidence interval for the standard deviation as output[4] and output[5], else output[4]=output[5]=0

def BiWeight.stddevbiweight (   vector,
  zero = None,
  eps = 1e-20 
)

PURPOSE: Estimation of the scale ('dispersion') of a distribution, resistant for outliers and robust for a broad range of non-Gaussian underlying populations.

INPUT: vector = distribution in vector form

OUTPUT: stddev = scale of the distribution, -1 in case of failure

KEWORDS: zero = set this keyword to force the scale to be calculated w.r.t. zero central location, e.g. in the case the vector elements are residuals of some fit esp = small number: zero scale if median absolute deviation (MAD) < EPS
