00109 ## @mainpage Gravlite - Non-Parametric Mass Modelling Routine for Discs and Spheres
00110 #
00111 # @section intro_sec Introduction
00112 #
00113 # Gravlite is a tool to determine the mass distribution in disc-like and spherical systems.
00114 # It takes as input a tracer density distribution, a line-of-sight velocity dispersion, and the velocity's fourth moment as a function of radius.
00115 # It then generates a highly-dimensional parameter space for tracer density, overall density distribution, and possibly a velocity anisotropy profile ands traces it with a simple Markov Chain Monte Carlo method.
00116 #
00117 # @section install_sec Installation
00118 # Following packets need to be installed on your system:
00119 # * python3
00120 # * matplotlib/pylab
00121 # * scipy
00122 #
00123 # Unzip the file gravlite.zip, and run
00124 #
00125 # python3 gravlite.py
00126 # 
00127 # @subsection pars Parameter files: Main configuration
00128 # Sample parameter files for several possible scenarios are stored in the subfolder ./params.
00129 # The file gl_params.py is a soft link to one of them.
00130 # Following mass modelling methods have been implemented so far:
00131 # * gl_params_walker.py: spherical Walker mock data from the Gaia challenge catalogue, 2 populations
00132 # * gl_params_gaia.py:   spherical mock data from the Gaia challenge catalogue, 1 population
00133 # * gl_params_hern.py:   spherical mock data taken from a Hernquist profile
00134 # * gl_params_simple.py: disk-like mock data, generated on the fly
00135 # * gl_params_sim.py:    disk-like mock data, from a simulation by S. Garbari
00136 # 
00137 # @subsection init Init values
00138 # Initial values for all profiles and the respective stepsizes are set in gl_init.py. We use following representations:
00139 # * overall density rho: can either be set as rho_i=rho(r_i), where r_i are the radii of bins, or the coefficients of the Legendre polynomial;
00140 # * tracer densities nu_i = nu(r_i) in linear space or nu_i = log(nu(r_i)) in logarithmic space;
00141 # * velocity anisotropy beta_i are incremental changes from beta(r=0) = 0.
00142 # 
00143 # Have fun!
