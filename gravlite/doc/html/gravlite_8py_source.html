111 # @section intro_sec Introduction
112 #
113 # Gravlite is a tool to determine the mass distribution in disc-like and spherical systems.
114 # It takes as input a tracer density distribution, a line-of-sight velocity dispersion, and the velocity's fourth moment as a function of radius.
115 # It then generates a highly-dimensional parameter space for tracer density, overall density distribution, and possibly a velocity anisotropy profile ands traces it with a simple Markov Chain Monte Carlo method.
