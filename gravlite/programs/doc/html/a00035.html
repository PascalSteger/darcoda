Detailed Description

parameters for MultiNest.

cube = [0,1]^ndim, ndim = nepol + 2*pops*(nepol+nbeta), Holds representations for overall density, [tracer density, anisotropy]_{for each population} Gives access to density, population profiles, and calculate physical values from [0,1] populations are counted from 0 = first component, 1 = first additional component disc version, done
